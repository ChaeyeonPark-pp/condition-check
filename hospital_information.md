병원 안내
<기능>
:병원의 거리 순으로 알려주는 게 아니라 현재 발생한 증상에 따라 필수적인 진료 기능을 갖춘 병원을 우선적으로 알려준다.

응급상황 선택시 - 응급실 운영 여부 / 소아 전문 응급 의료기관 여부
내상 선택시 - 소아청소년과 / 해당 질환 가능 여부
외상 선택시 - 정형외과 , 외과 등 외상 처치 가능 여부

현재 운영 상태 정보 연동: 병원위치와 함께 현재 시점 진료가능한지(야간,공휴일)를 실시간으로 표시하도록 함

<위치기반 서비스 활용>
사용자 위치 자동파악: 현재 휴대폰의 위치를 정확히 파악해 가장 가까운 병원을 기준으로 목록 제공
내비게이션 연동: 검색된 병원을클릭하면 스마트폰의 기본 내비게이션앱(카카오내비,T맵,구글맵)으로 바로 길 안내를 시작할 수 있도록 연동 기능 제공

<병원 도착 전 정보 제공>
병원 정보 상세 보기:안내되는 병원목록에서 전화번호, 진료시간, 의사전문 분야 등 상세정보를 바로 확인할 수 있도록 한다.
전화연결 기능: 병원 정보에서 전화버튼을 제공해 병원도착 전에 현재 아이상태를 미리 전달하거나 대기시간 여부를 확인할 수 있도록 돕는다.


사용방법
119
예상치못한 응급상황에 대비하여 119버튼 생성하여 위기 대처 목적
응급상황 / 내상 / 외상 오른쪽 하단에 119 버튼 있음
119 버튼을 누르면 실수로 눌렀을 때를 대비하여 진짜 119를 부를건지 한번 더 체크

약품
당장 가지고 있는 약품으로 아이 증상을 해결할 수 있는지 없는지 확인,
부모가 용량을 잘못 계산하는 위험 줄임 목적

대처방법 밑에 [약품 사용 가능 여부 판단하기] 버튼 생성
버튼을 누를 시 추천한 약 존재 여부 묻기
약이 있을 시 사용 체크리스트 생성
없을 시 위치기반서비스로 병원/약국 추천
