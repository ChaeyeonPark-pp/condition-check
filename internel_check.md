1. 내상 상태 페이지

1. 탈수
증상: 입 마름, 눈물 없음, 기저귀 안 젖음, 무기력
대처 방법: 전해질 수액 섭취

대표적인 전해질 물질 : 나트륨, 칼륨, 칼슘, 마그네슘, 염소, 중탄산염  
전해질 음료 : 포카리스웨트, 게토레이(물로 희석 후 섭취)
약국 : 오알에스
약국용 : 디오라이트 (구토, 설사 후 전해질 보충용)

주의 : 과도한 당분이 들어간 (탄산음료, 과일주스)는 설사 악화시킬 수 있으니 자제.
집에서 만드는 ORS : 깨끗한 물 1리터(생수나 끓였다 식힌 물)
                   소금 작은술 (1/2) (약 3g)
                   설탕 작은술 (6) (약 30g)

2. 장염
증상: 구토, 설사, 복통, 발열
대처 방법: 음식 섭취 중지 및 수분 섭취, 24시간 이상 지속 시 병원 방문
섭취 가능한 수분 예시 : 전해질이 포함된 적절한 수분

전해질 음료 : 포카리스웨트, 게토레이(물로 희석 후 섭취)
약국 : 오알에스
집에서 만드는 ORS : 깨끗한 물 1리터(생수나 끓였다 식힌 물)
                   소금 작은술 (1/2) (약 3g)
                   설탕 작은술 (6) (약 30g)

3. 호흡기 감염
증상: 기침, 숨 가쁨, 열, 산소 부족
대처 방법: 수분 섭취, 가습, 증상 악화 시 병원
섭취 가능한 수분 예시 : 물, 미지근한 물, 따뜻한 차 혹은 국물
가습 정도 : 실내가 건조해질 때 가습기 사용
증상 악화 정도 : 아이가 숨 쉬기를 꽤나 힘들어 할 경우, 아이가 먹거나 마시기를 거부하는 경우, 소변량이 급격히 줄어들 경우, 의식이 흐릿할 경우

섭취 가능한 수분 예시 : 물, 미지근한 물, 보리차 따뜻한 국물, 꿀물(1세 이상)
4. 중이염
증상: 귀를 자주 만짐, 울음, 발열
대처 방법: 통증 시 따뜻한 찜질
증상 구체적 예시 : 아이가 갑자기 귀를 자꾸 잡아당기거나 문지를 경우, 평소보다 자주 손이 귀 쪽으로 가져갈 경우, 귀 뒤를 만질 때 불편한 표정을 지을 경우 > 습관적 행동이 아닌 " 통증 반응으로 보이는 행동 " 일 경우
울음의 기준 : 잠들기 전 누웠을 때 울음이 심해질 경우, 귀 건드릴 때만 심하게 움찔하거나 울었을 경우

대처 시 주의해야 될 점 : 손으로 만졌을 때 따뜻하다 (약 40 ~ 45도) 일 때

깨끗한 수건을 사용할 경우 : 뜨거운 물에 적신 뒤, 물기를 없애고 귀 주변에 댄다
전용 온찜질팩 : 사용법 그대로 사용
핫팩(응급용) : 직접 피부에 닿지 않게 수건 2겹 이상 감싸서 사용

귀 안에 넣는 게 아니라 귓바뮈와 귀 뒤쪽(귓불 뒤)에 대야 함
1회 5~10분, 하루 2~3회 정도면 충분함.

주의해야 될 점 : 열 38.5도면 찜질 금지
                고름이나 분비물이 나오는 경우도 금지
                > 고름이나 분비물이 나올 경우 이미 고막이 손상 된 상태일 가능성 있어 바로 병원 가야 함

                찜질 후 아이 귀가 붉거나 따가워하면 즉시 중단

5. 변비
증상: 배변시 울음, 복부 팽만
대처 방법: 수분과 채소 섭취 증가
섭취해야 될 수분 구체적 예시 : 미지근한 물, 카페인 없는 보리차, 보리물 가능
변비 완화에 좋은 채소 예시 : 시금치, 브로콜리, 애호박, 단호박

복부 팽만 예시 : 배변 시 울거나, 배가 단단하게 만져짐, 변이 딱딱하게 굵음 ( 평소보다 배변 간격이 길어지고, 하루 이상 변이 안 나올 때가 많음) 
아이의 배를 세개 누를 때 심하게 울거나 움찔할 경우
복부가 매우 단단하고 가스가 빠지지 않을 때
토하거나 구토에 변 냄새가 날 때
열, 식욕 저하, 무기력이 동반될 때의 경우 
> 이럴 경우 병원 가야됨.

대처 방법 : 복부 팽만의 경우 아이를 눕히고 배꼽을 중심으로 시계 방향으로 부드럽게 3 ~ 5분 마사지 > 장운동 촉진 도움
