1. 내상 상태 페이지

내상 대표적인 종류
1. 탈수 
증상 : 입 마름, 눈물 없음, 기저귀 안 젖음, 무기력
대처 방법 : 전해질 수액 섭취
2. 장염
증상 : 구토, 설사, 복통, 발열
대처 방법 : 음식 섭취 중지 및 수분 섭취, 24시간 이상 지속 시 병원 방문
3. 호흡기 감염
증상 : 기침, 숨 가끔, 열, 산소 부족
대처 방법 : 수분 섭취, 가습, 증상 악화 시 병원
4. 중이염
증상 : 귀를 자주 만짐, 울음, 발열
대처 방법 : 통증 시 따뜻한 찜질
5. 변비
증상 : 배변시 울음, 복부 팽만
대처 방법 : 수분과 채소 섭취 증가
